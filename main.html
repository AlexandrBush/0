Welcome to the User Management System

This is the main page of the application. Here you can manage users and view their details.

Navigation

  • View All Users
  • View User Details (Example: User ID 1)

How to Use

- To view the list of all users, click on the "View All Users" link.
- To view details of a specific user, click on the "View User Details" link and replace "1" with the desired user ID.
